# Quickstart

The fastest way to try Vortex is with the `vx` command-line tool. In three steps you can
convert a Parquet file, explore its structure, and run SQL queries — all from the terminal.

```{toctree}
---
maxdepth: 2
caption: vx CLI
---

install
convert
query
```
